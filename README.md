# courses-frontend

Frontend simple para taller: HTML + CSS + JavaScript (sin frameworks).

## Páginas

- `static/index.html`: landing inicial estática
- `static/course-1.html`: detalle estático curso 1
- `static/course-2.html`: detalle estático curso 2
- `static/course-3.html`: detalle estático curso 3
- `static/courses.json`: datos base de los 3 cursos (editable para clase)
- `index.html`: home público (dinámico)
- `course.html?id=<id>`: detalle dinámico
- `login.html`: login admin
- `admin-courses.html`: CRUD + activar/desactivar

## Enfoque para clase

- Se redujo la complejidad de scripts para enseñanza inicial.
- Se eliminó la función `escapeHtml` y varias validaciones avanzadas.
- Se mantiene el mismo diseño visual en la versión estática y dinámica.

## Requisitos

- Backend corriendo en `http://localhost:3000`

## Ejecución

Puedes abrir los HTML directamente o usar un servidor estático local.

Ejemplo con VS Code Live Server o cualquier servidor simple.
