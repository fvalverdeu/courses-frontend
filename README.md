# courses-frontend

Frontend simple para taller: HTML + CSS + JavaScript (sin frameworks).

## Páginas

- `index.html`: home público (dinámico)
- `course.html?id=<id>`: detalle del curso
- `login.html`: login admin
- `admin-courses.html`: CRUD + activar/desactivar

## Requisitos

- Backend corriendo en `http://localhost:3000`

## Ejecución

Puedes abrir los HTML directamente o usar un servidor estático local.

Ejemplo con VS Code Live Server o cualquier servidor simple.
